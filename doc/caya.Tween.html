Class: Tween

caya.Tween(from, to, duration, onUpdateopt, onDoneopt, easefopt, procfopt)

Class used for tweening values.

Constructor

new Tween(from, to, duration, onUpdateopt, onDoneopt, easefopt, procfopt)

Constructs a Tween object.
Parameters:
Name Type Attributes Default Description
from number Value to tween from.

onUpdate tweenUpdateCallback <optional>
 Update callback.
onDone function <optional>
 Triggers when Tween finishes.
easef function <optional>
 caya.ease.linear Easing function. Use one of the functions from caya.ease (caya.ease.linear, caya.ease.quadIn, caya.ease.quadOut, caya.ease.quadInOut, caya.ease.backIn or caya.ease.backOut) or substitute your own.

 Value post-process function.

Examples
// Tween from 0 to 100 over 1.5 seconds with quadInOut easing
(new caya.Tween(0, 100, 1500, function(value) {
   console.log(value);
}, function() {
   console.log('Tween done!');
}, caya.ease.quadInOut)).start();
// Tween from 100 to 0 over 2 seconds with linear easing and floor output values.
(new caya.Tween(100, 0, 2000, function(value) {
   console.log(value);
}, function() {
   console.log('Tween done!');
}, caya.ease.linear, Math.floor)).start();
